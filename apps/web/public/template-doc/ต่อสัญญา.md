|  โดยทั้งสองฝ่ายจึงได้ตกลงกันดังต่อไปนี้ 1\. ขยายวันสิ้นสุดของบันทึกข้อตกลงต่อสัญญาเช่าฉบับลงวันที่ \<\<สัญญาเช่าฉบับเก่าลงวันที่\>\> จากเดิมสิ้นสุดวันที่ \<\<สิ้นสุดสัญญาวันที่\>\> เป็น \<\<ขยายเวลาสิ้นสุดเป็นวันที่\>\>          2\. ข้อกำหนดและเงื่อนไขอื่นๆที่ระบุไว้ในสัญญาฉบับลงวันที่  \<\<สัญญาเช่าฉบับเก่าลงวันที่\>\> ยังคงมีผลบังคับใช้เหมือนสัญญาเช่าฉบับเดิมทุกประการ และให้ตีความและบังคับใช้ตามกฎหมายไทย 3\. ค่าเช่า ทรัพย์สินเป็นจำนวนเงินเดือนละ  \<\<ค่าเช่า\>\> บาท ต่อเดือน ( \<\<ค่าเช่าบาท\>\> ) สำหรับระยะเวลาเช่า กำหนดไว้เป็น \<\<ระยะเวลาต่อสัญญา\>\> เดือน โดยเริ่มสัญญาวันที่ \<\<เริ่มต่อสัญญา\>\> สิ้นสุดวันที่ \<\<ขยายเวลาสิ้นสุดเป็นวันที่\>\> บันทึกข้อตกลงฉบับนี้ทำขึ้น 2 ฉบับ มีข้อความตรงกัน ผู้ให้เช่า และ ผู้เช่า ถือไว้คนละฉบับ  |  |  |  |  |  |  |  |  |
