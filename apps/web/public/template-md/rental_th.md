2.2. “ผู้เช่า” ได้ชำระค่าเช่าสำหรับเดือนแรกของสัญญาเรียบร้อยแล้วในวันที่เซ็นสัญญาฉบับนี้   
	2.3. “ผู้ให้เช่า” เป็นผู้ชำระค่าส่วนกลางที่เรียกเก็บโดยนิติบุคคลของโครงการที่ “ทรัพย์สิน” นี้ ตั้งอยู่  
	2.4. หาก ผู้เช่า ไม่ได้ชำระค่าเช่าและพ้นระยะกำหนดชำระเกิน 15 วัน ให้ถือว่า ผู้เช่า ละเมิดสัญญาเช่า   
ผู้ให้เช่า มีสิทธิ์ยกเลิกสัญญาเช่า และ เรียกค่าเสียหายล่วงหน้าจากการผิดสัญญาเช่าก่อนกำหนด และ ผู้เช่า ต้องย้ายออกจาก ทรัพย์สิน หลังจากได้รับหนังสือยกเลิกสัญญาเช่าเป็นลายลักษณ์อักษร ภายใน 7 วัน และ ผู้ให้เช่า มีสิทธิ์กลับเข้าครอบครอง ทรัพย์สิน ตามเดิม  
**ข้อ 3\. รายละเอียดเงินประกันสัญญา**

| 3.1. ในวันที่เซ็นสัญญาฉบับนี้ “ผู้เช่า” ได้ชำระเงินให้แก่  “ผู้ให้เช่า” เป็นจำนวน |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  | \<\<จำนวนเงินวันทำสัญญา\>\> บาท |  |  |  |

| (\<\<จำนวนเงินวันทำสัญญาตัวอักษร\>\>) |  |  |  |  |  |  |  |  ซึ่งต่อไปนี้เรียกว่า “เงินประกันสัญญา” ซึ่งเงินจำนวนดังกล่าว |  |  |  |  |  |  |  |  |  |  |  |

“ผู้ให้เช่า” จะเก็บไว้ตลอดอายุของสัญญาโดยไม่มีดอกเบี้ยใดๆ เพื่อเป็นประกันความเสียหายที่อาจจะเกิดขึ้นแก่   
“ทรัพย์สิน” ที่อยู่ในความรับผิดชอบของ “ผู้เช่า” และหรือบริวารของ “ผู้เช่า” และเพื่อเป็นหลักประกันแก่ความเสียหายที่อาจเกิดขึ้นในกรณีที่ “ผู้เช่า” ละเมิดข้อตกลงที่ระบุไว้ในสัญญาฉบับนี้  
3.2. “ผู้ให้เช่า” จะชำระคืน “เงินประกันสัญญา” ให้แก่ “ผู้เช่า” หลังจากได้หักหนี้ค้างชำระใดๆ ที่เป็นความรับผิดชอบของผู้เช่าภายใน  30 วัน นับจากวันสิ้นสุดสัญญา   
3.3. “ผู้ให้เช่า” ได้มอบใบเสร็จรับเงินสำหรับ เงินประกันสัญญา ให้กับ “ผู้เช่า” ไว้แล้ว  
3.4. เงินประกันสัญญา ไม่สามารถนำมาชำระแทนค่าเช่า ค่าน้ำ ค่าไฟ และค่าบริการอื่นๆ ได้  และ “ผู้เช่า” ไม่สามารถนำมาเป็นข้ออ้างในการไม่ชำระค่าเช่าได้  
3.5. สัญญาฉบับนี้จะสมบูรณ์ต่อเมื่อ “ผู้ให้เช่า” ได้รับเงินค่าเช่าตามข้อ 2.1 และ เงินประกันสัญญา ตามข้อ 3.1 จาก “ผู้เช่า” อย่างครบถ้วน เรียบร้อยแล้ว

**ข้อ 4\. ข้อตกลงและข้อปฏิบัติของผู้เช่า**  
“ผู้เช่า” สัญญาว่าจะปฏิบัติตามข้อตกลงดังนี้  
4.1. ชำระค่าเช่า ค่าสาธารณูปโภคอื่นๆ ให้ครบถ้วนตรงตามวันที่ระบุไว้ในสัญญาฉบับนี้ อันได้แก่ ค่าไฟฟ้า ค่าน้ำประปา  ค่าโทรศัพท์ และอื่นๆ ที่เกิดขึ้นจากการใช้งานของ ผู้เช่า ให้ครบถ้วนตรงตามกำหนด  ตลอดอายุสัญญา การละเลยไม่ชำระค่าสาธารณูปโภคดังกล่าวและเป็นเหตุให้เกิดความเสียหายแก่ ผู้ให้เช่า หรือ ทรัพย์สิน สามารถถือเป็นการละเมิดสัญญาฉบับนี้ได้  
4.2. ปฏิบัติตามข้อตกลงที่ระบุไว้ในสัญญาฉบับนี้ตลอดจนข้อกำหนดของนิติบุคคลของโครงการ ซึ่งเกี่ยวข้องกับการใช้อาคารและ ทรัพย์สิน  
4.3. ใช้ ทรัพย์สิน เพื่อการอยู่อาศัยเท่านั้น  
4.4. ดูแลรักษาส่วนประกอบของ ทรัพย์สิน อันประกอบด้วย พื้น ผนัง เพดาน หน้าต่าง ประตู เฟอร์นิเจอร์ และอุปกรณ์ไฟฟ้า รวมทั้งส่วนที่ยึดติดอยู่กับ ทรัพย์สิน ให้อยู่ในสภาพเรียบร้อย ปราศจากความเสียหาย ยกเว้นความเสียหายที่เกิดขึ้นจากการใช้งานปกติ หรือหมดอายุการใช้งานของส่วนประกอบนั้นๆ ตลอดอายุสัญญา  
4.5. อนุญาตให้ “ผู้ให้เช่า” หรือตัวแทนเข้าตรวจสภาพ “ทรัพย์สิน” ในเวลากลางวัน โดยต้องทำการแจ้งล่วงหน้าไม่น้อยกว่า 24 ชั่วโมงและต้องได้รับความยินยอมจาก ผู้เช่า ด้วยเช่นกัน  
4.6. ก่อนสิ้นสุดสัญญาเช่า 30 วัน “ผู้เช่า” อนุญาตให้ “ผู้ให้เช่า” หรือตัวแทน เปิดให้ชม “ทรัพย์สิน” แก่ผู้จะเช่าหรือผู้จะซื้อรายอื่น โดยต้องทำการแจ้งล่วงหน้าไม่น้อยกว่า 24 ชั่วโมงและต้องได้รับความยินยอมจาก “ผู้เช่า” ด้วยเช่นกัน  
4.7. ไม่ต่อเติม ดัดแปลง หรือเปลี่ยนแปลง ส่วนหนึ่งส่วนใดของ “ทรัพย์สิน” โดยไม่ได้รับความยินยอมจาก “ผู้ให้เช่า” เป็นลายลักษณ์อักษร หาก “ผู้เช่า” กระทำการฝ่าฝืน “ผู้เช่า” จะต้องรับผิดชอบต่อความเสียหายที่เกิดขึ้นจากการกระทำดังกล่าว  
4.8. ในกรณีหากมีสิ่งต่อเติม ดัดแปลงใดๆ ได้ทำการยึดติดกับ “ทรัพย์สิน” ให้ถือว่าเป็นส่วนหนึ่งของ   
“ทรัพย์สิน” และให้ถือเป็นกรรมสิทธิ์ของ “ผู้ให้เช่า” เมื่อสัญญาเช่าสิ้นสุดลง  
4.9. ไม่นำส่วนหนึ่งส่วนใดหรือทุกส่วนของ “ทรัพย์สิน” ไปใช้เพื่อวัตถุประสงค์ที่ผิดต่อศีลธรรมอันดีงามหรือผิดกฎหมาย  
4.10. ไม่ปล่อยเช่าช่วง “ทรัพย์สิน” หรือถ่ายโอนสิทธิตามที่ระบุไว้ในสัญญาฉบับนี้ให้แก่ผู้อื่น โดยไม่ได้รับความยินยอมจาก “ผู้ให้เช่า” เป็นลายลักษณ์อักษร   
4.11. ระหว่างที่พักอาศัยอยู่ใน “ทรัพย์สิน” “ผู้เช่า” ต้องไม่กระทำการอันใดที่ก่อให้เกิดความไม่สงบ ก่อกวน หรือล่วงละเมิดใดๆ แก่ผู้ที่พักอาศัยในบริเวณใกล้เคียง  
4.12. แจ้งความเสียหาย ความบกพร่องที่เกิดขึ้นกับ “ทรัพย์สิน” ให้กับ “ผู้ให้เช่า” ทราบในทันทีที่พบความเสียหาย ความบกพร่องดังกล่าว และต้องไม่เกินกว่า 7 วัน นับจากวันที่เกิดความเสียหาย ความบกพร่องแก่ “ทรัพย์สิน”  
4.13 ความบกพร่องแก่ ทรัพย์สิน ในระหว่างการเช่า หลังจาก 7 วันที่ย้ายเข้าอยู่อาศัย หากมีสิ่งใดชำรุด หรือ เสียหาย เช่น หลอดไฟเสีย , สายชำระ อื่นๆ  “ผู้เช่า” ต้องเป็นฝ่ายรับผิดชอบ ค่าใช้จ่ายในวงเงินไม่เกิน 1,000 บาท ของการจัดการซ่อม  
4.14. หากความเสียหายที่เกิดขึ้นกับ “ทรัพย์สิน” เป็นเหตุจาก “ผู้เช่า” หรือบริวารของ “ผู้เช่า” “ผู้เช่า” จะต้องทำการซ่อมแซมความเสียหายที่เกิดขึ้นกับ “ทรัพย์สิน” เพื่อให้อยู่ในสภาพใช้งานได้เหมือนเดิม  
4.15. ส่งมอบกุญแจทั้งหมดที่เกี่ยวข้องกับ “ทรัพย์สิน” ให้แก่ “ผู้ให้เช่า” ในวันที่สัญญาเช่าสิ้นสุดลง  
4.16. ไม่อนุญาตให้นำสัตว์เลี้ยงเข้ามาเลี้ยง หรือ อาศัยอยู่ใน “ทรัพย์สิน”  
4.17. ห้ามมิให้มีการ สูบบุหรี่ ภายในห้องเป็นอันขาด หากเกิดการพบเจอ หรือ ตรวจจับได้ว่ามีการสูบบุหรี่ภายในห้องตามที่ระบุไว้ในข้อนี้ “ผู้ให้เช่า” สามารถบอกเลิกสัญญา โดยแจ้งเป็นลายลักษณ์อักษรแก่ “ผู้เช่า” ล่วงหน้าไม่น้อยกว่า 7 วัน โดยส่งมายังที่อยู่ของ “ผู้เช่า” ที่ระบุไว้ในสัญญาฉบับนี้ หรือส่งมายังที่อยู่ของ “ทรัพย์สิน” โดยตรง และ “ผู้เช่า” ต้องชำระค่าเสียหายตามจริง ที่เกิดขึ้นจากการสูบบุหรี่ภายในห้องเช่า และ “ผู้ให้เช่า” มีสิทธิกลับเข้าครอบครอง “ทรัพย์สิน” โดยทันทีหลังระยะเวลาบอกกล่าวดังกล่าวสิ้นสุดลง

**ข้อ 5\. ข้อตกลงและข้อปฏิบัติของผู้ให้เช่า**  
“ผู้ให้เช่า” สัญญาว่าจะปฏิบัติตามข้อตกลงดังนี้  
5.1. “ผู้ให้เช่า” รับประกันว่าเป็นผู้มีอำนาจปล่อยให้เช่า “ทรัพย์สิน” โดยถูกต้อง  
5.2. หาก  “ผู้เช่า” ชำระค่าเช่าและปฏิบัติตามข้อตกลงในสัญญาฉบับนี้อย่างครบถ้วน “ผู้ให้เช่า” หรือตัวแทน จะไม่กระทำการใดๆ ที่เป็นการก่อให้เกิดความไม่สงบ ก่อกวน หรือล่วงละเมิดใดๆแก่ “ผู้เช่า” ตลอดอายุสัญญา  
5.3. “ผู้ให้เช่า” ต้องบำรุงรักษา “ทรัพย์สิน”  ให้อยู่ในสภาพพักอาศัย และใช้งานได้ตามปกติ รวมทั้งต้องซ่อมแซมส่วนที่เสียหายที่ไม่ได้เกิดจากการใช้งาน “ทรัพย์สิน”  ผิดลักษณะของ “ผู้เช่า” โดยทันทีที่ได้รับแจ้ง

**ข้อ 6\. การบอกเลิกสัญญา**  
6.1. “ผู้ให้เช่า” สามารถบอกเลิกสัญญาฉบับนี้ได้ โดย “ผู้เช่า” ไม่มีสิทธิโต้แย้ง หากเกิดเหตุการณ์ต่อไปนี้  
a).”ผู้เช่า” ไม่ชำระค่าเช่าหรือค่าใช้จ่ายอื่นๆ ให้ครบถ้วน ตรงตามเวลาที่ระบุไว้ในสัญญาฉบับนี้  
b) “ผู้เช่า” ฝ่าฝืนหรือไม่ปฏิบัติตามข้อตกลงข้อหนึ่งข้อใดที่ระบุไว้ในสัญญาฉบับนี้  
c) “ผู้เช่า” เป็นบุคคลล้มละลายตามกฎหมาย  
d) หากส่วนหนึ่งส่วนใด หรือทุกส่วนของ “ทรัพย์สิน” ไม่สามารถใช้พักอาศัยได้ อันเนื่องจาก อัคคีภัย คำสั่งทางกฎหมาย หรือการจัดสรรเขตของทางรัฐบาล  
e) ภัยพิบัติทางธรรมชาติ อาทิเช่น แผ่นดินไหว สึนามิ เป็นต้น เป็นเหตุให้ทรัพย์สินไม่สามารถอยู่อาศัยได้ตามเอกสารรับรองของเจ้าหน้าที่ประเมิน หรือ หน่วยงานที่เกี่ยวข้อง  
6.2. หากเกิดเหตุการณ์ใดเหตุการณ์หนึ่งตามที่ระบุไว้ในข้อ 6.1 “ผู้ให้เช่า” สามารถบอกเลิกสัญญา โดยแจ้งเป็นลายลักษณ์อักษรแก่ “ผู้เช่า” ล่วงหน้าไม่น้อยกว่า 7 วัน โดยส่งมายังที่อยู่ของ “ผู้เช่า” ที่ระบุไว้ในสัญญาฉบับนี้ หรือส่งมายังที่อยู่ของ “ทรัพย์สิน” โดยตรง และมีสิทธิกลับเข้าครอบครอง “ทรัพย์สิน” โดยทันทีหลังระยะเวลาบอกกล่าวดังกล่าวสิ้นสุดลง  
6.3. หากการบอกเลิกสัญญาเกิดขึ้นจากเหตุการณ์ในข้อ 6.1 (a), (b) หรือ (c) “ผู้ให้เช่า” มีสิทธิริบ เงินประกันสัญญา ที่ระบุไว้ในข้อ 3.1 รวมทั้งค่าเช่าที่รับชำระไว้แล้ว ทั้งยังสามารถเรียกร้องค่าเสียหายที่อาจเกิดขึ้นจากการบอกเลิกสัญญาฉบับนี้ได้  
6.4. หากการบอกเลิกสัญญาเกิดขึ้นจากเหตุการณ์ในข้อ 6.1 (d) และ (e) ทั้งสองฝ่ายไม่มีสิทธิเรียกร้องค่าเสียหายใดๆจากอีกฝ่ายหนึ่งได้ และ “ผู้ให้เช่า” ต้องคืน เงินประกันสัญญา ที่ระบุไว้ในข้อ 3.1 หลังจากหักค่าใช้จ่ายค้างชำระที่ “ผู้เช่า” เป็นผู้ต้องรับผิดชอบแล้ว ภายใน 30 วัน หลังจากวันที่สัญญาสิ้นสุดลง   
6.5. หาก “ผู้เช่า” ต้องการบอกเลิกสัญญา หรือต่อสัญญา ต้องแจ้งเป็นลายลักษณ์อักษรต่อ “ผู้ให้เช่า” เป็นระยะเวลาไม่น้อยกว่า 30 วัน ก่อนวันสิ้นสุดสัญญา หากไม่มีการแจ้งใดๆจาก “ผู้เช่า” ให้ถือว่า “ผู้ให้เช่า” สามารถหาผู้เช่ารายใหม่ได้ภายใต้เงื่อนไขข้อ 4.6 และ 4.7 ของสัญญานี้   
6.6. หลังจากแจ้งการบอกเลิกหรือการไม่ต่อสัญญาเป็นลายลักษณ์อักษรแล้ว ในระหว่างระยะเวลา 30 วันก่อนวันสิ้นสุดสัญญา “ผู้เช่า” ยินยอมให้ “ผู้ให้เช่า” นำ “ผู้จะเช่า” หรือ “ผู้จะซื้อ” รายใหม่เข้าชม “ทรัพย์สิน” ได้โดยแจ้งล่วงหน้าเป็นเวลาไม่น้อยกว่า 24 ชั่วโมง  
6.7. หาก “ผู้เช่า” บอกเลิกสัญญาก่อนวันสิ้นสุดสัญญาที่ระบุไว้ในข้อ 1 จะต้องแจ้งเป็นลายลักษณ์อักษรแก่ “ผู้ให้เช่า”   ล่วงหน้าเป็นเวลาไม่น้อยกว่า 30 วันและ “ผู้ให้เช่า” มีสิทธิริบเงินประกันการเช่าทั้งหมด เพื่อเป็นค่าเสียหายล่วงหน้าจากการเสียโอกาสในการให้เช่าและค่าใช้จ่ายในการบริหารจัดการทรัพย์สิน ตามข้อ 3.1

**ข้อ 7\. ข้อตกลงเรื่องการย้ายออกจากทรัพย์สิน**  
7.1. ในวันที่ครบกำหนดสัญญาเช่า “ผู้เช่า” จะต้องย้ายและขนย้ายทรัพย์สินที่เป็นของ “ผู้เช่า” ออกจาก “ทรัพย์สิน” และส่งมอบ “ทรัพย์สิน” คืนแก่ “ผู้ให้เช่า” และ “ผู้เช่า” ไม่มีสิทธิเรียกร้องค่าขนย้ายใดๆจาก “ผู้ให้เช่า”  
7.2. หาก “ผู้เช่า” ไม่สามารถทำตามข้อตกลงที่ระบุไว้ข้อ 7.1 แล้ว “ผู้เช่า” ตกลงว่า  
a) ให้ถือว่าทรัพย์สินของ “ผู้เช่า” ที่ยังคงอยู่ภายใน “ทรัพย์สิน” ตกเป็นกรรมสิทธิ์ของ “ผู้ให้เช่า” และ “ผู้ให้เช่า” มีสิทธิที่จะดำเนินการใดๆกับทรัพย์สินดังกล่าวได้ หรือสามารถเรียกร้องค่าเสียหายที่อาจเกิดขึ้น หากทรัพย์สินของ “ผู้เช่า” ดังกล่าวไม่ได้รับการโอนให้เป็นกรรมสิทธิ์ของ “ผู้ให้เช่า” หรือสร้างความเสียหายแก่ “ทรัพย์สิน” ได้  
b) กรณีที่ไม่ย้ายออกตามกำหนดระยะเวลาของสัญญา “ผู้เช่า” ต้องชำระค่าเช่าให้แก่ “ผู้ให้เช่า” เป็นจำนวนเท่ากับ ครึ่งหนึ่งของค่าเช่ารายเดือน หาก “ผู้เช่า” ไม่สามารถทำตามข้อตกลงในข้อ 7.1 ได้ภายใน 15 วันนับแต่วันที่หมดสัญญา และต้องชำระค่าเช่าเป็นจำนวนเท่ากับค่าเช่าหนึ่งเดือน หาก “ผู้เช่า” ไม่สามารถทำตามข้อตกลงในข้อ 7.1 ได้ แต่ใช้เวลาเกินกว่า 15 วัน และไม่เกิน 30 วัน นับแต่วันที่หมดสัญญา รวมทั้งชำระค่าปรับให้แก่ “ผู้ให้เช่า” จำนวนวันละ 1,000.-  บาท (หนึ่งพันบาทถ้วน) นับจากวันที่หมดสัญญา
